## Task-Specific Workflows

### Workflow 1: Product Launch Copy
1. **Brief Intake**
   - Review product details and features
   - Identify target audience segments
   - Confirm key messages and positioning
   - Note launch timeline

2. **Messaging Development**
   - Craft core value proposition
   - Develop supporting benefit statements
   - Create emotional hooks
   - Align with brand voice

3. **Copy Creation**
   - Write product headlines (5-10 variations)
   - Draft product descriptions (short + long)
   - Create CTA variations
   - Develop social proof integration

4. **Format Adaptation**
   - Adapt for web product page
   - Create social media versions
   - Draft email announcement copy
   - Prepare ad copy variations

5. **Output**
   - Save to: `workspace/memory/night-shift/content/product-[name]-YYYY-MM-DD.md`
   - Include all variations
   - Note recommended usage

### Workflow 2: Social Media Content
1. **Topic Identification**
   - Check Trend-Analyzer outputs for timely topics
   - Review social monitoring data
   - Identify engagement opportunities

2. **Platform-Specific Drafting**
   - Instagram: Visual-first captions, hashtags
   - LinkedIn: Professional insights, longer form
   - Twitter/X: Concise, thread potential
   - TikTok: Trend-aware, casual

3. **Content Variations**
   - Create 3-5 caption options per post
   - Draft hashtag sets (branded + trending)
   - Prepare engagement prompts
   - Write alt text for accessibility

4. **Scheduling Prep**
   - Recommend optimal posting times
   - Note cross-platform sequencing
   - Flag time-sensitive content

5. **Output**
   - Save to: `workspace/memory/night-shift/content/social-YYYY-MM-DD.md`
   - Organize by platform and date

### Workflow 3: Marketing Email Content
1. **Campaign Brief Review**
   - Understand email objective
   - Identify target segment
   - Confirm offer or CTA

2. **Subject Line Creation**
   - Write 10+ subject line variations
   - Include preview text options
   - Note personalization opportunities

3. **Body Copy Drafting**
   - Hook/lead paragraph
   - Benefit-focused body
   - Clear CTA placement
   - P.S. line options

4. **Segmentation Adaptation**
   - Create variations for different audiences
   - Adjust tone for customer lifecycle stage
   - Personalize with dynamic content markers

5. **Output**
   - Save to: `workspace/memory/night-shift/content/email-[campaign]-YYYY-MM-DD.md`

### Workflow 4: Campaign Messaging
1. **Campaign Strategy Review**
   - Understand campaign objectives
   - Identify key themes and pillars
   - Review competitive landscape

2. **Message Architecture**
   - Develop primary message
   - Create supporting messages
   - Build message hierarchy
   - Ensure consistency across touchpoints

3. **Channel Adaptation**
   - Web copy
   - Social copy
   - Email copy
   - Ad copy

4. **Tone Calibration**
   - Adjust for channel norms
   - Maintain brand voice
   - Optimize for audience

5. **Output**
   - Save to: `workspace/memory/night-shift/content/campaign-[name]-YYYY-MM-DD.md`
