# AGENTS.md - Handoff-Preparer

## Workflow

### Collection Phase (00:00-07:30 CET)

#### Continuous Collection
As agents complete work, collect:
1. Monitor summaries (Social, Competitor, News)
2. Report outputs (Sales, Content, Analysis)
3. Content deliverables (Social, Newsletter)
4. System status (Health, Backup, Organization)
5. QA approvals

#### 06:00 CET - Begin Brief Compilation
1. **Gather All Outputs**
   - Check memory/night-shift/ directories
   - Review handoff folder
   - Collect final reports
   - Verify QA approvals

2. **Prioritize Information**
   - Flag critical alerts
   - Identify action items
   - Rank by urgency
   - Note dependencies

#### Content Integration
1. **Trend Highlights**
   - Summarize from Trend-Analyzer
   - Include velocity scores
   - Note relevance to brand

2. **Alert Summary**
   - List all critical alerts
   - Include context
   - Recommend actions

3. **Content Package**
   - List ready-to-publish content
   - Note pending approvals
   - Include optimization notes

### Review Phase (07:30-07:45 CET)
1. **Quality Check**
   - Verify completeness
   - Check accuracy
   - Confirm priorities
   - Review clarity

2. **Final Approval**
   - Get Quality-Checker sign-off
   - Confirm with Night-Coordinator
   - Finalize formatting

### Handoff Delivery (07:45 CET)
1. Save final brief
2. Notify day shift
3. Confirm receipt
4. Document handoff

### Output File
Path: `workspace/memory/night-shift/handoff/MORNING-BRIEF-YYYY-MM-DD.md`

## Dependencies
- All night shift agents (outputs)
- Quality-Checker (approval)
- Night-Coordinator (sign-off)
- Day shift contact

## Success Metrics
- All outputs collected
- Brief delivered by 07:45 CET
- Critical alerts highlighted
