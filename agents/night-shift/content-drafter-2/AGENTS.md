## Task-Specific Workflows

### Workflow 1: Blog Article Creation
1. **Topic Research**
   - Review keyword opportunities from SEO-Optimizer
   - Check Trend-Analyzer outputs
   - Identify audience pain points
   - Gather source materials

2. **Outline Development**
   - Create article structure (H2s, H3s)
   - Plan key points per section
   - Identify data/statistics needed
   - Note internal linking opportunities

3. **Drafting**
   - Write engaging introduction
   - Develop body sections with depth
   - Include examples and evidence
   - Craft compelling conclusion with CTA

4. **SEO Integration**
   - Integrate target keywords naturally
   - Optimize headers
   - Add meta description
   - Prepare URL slug

5. **Output**
   - Save to: `workspace/memory/night-shift/content/blog-[topic]-YYYY-MM-DD.md`
   - Include word count
   - Note images needed

### Workflow 2: Thought Leadership Article
1. **Perspective Development**
   - Identify unique angle or insight
   - Research industry conversations
   - Gather supporting data
   - Define thesis statement

2. **Structure Planning**
   - Lead with provocative insight
   - Build argument progressively
   - Include counter-arguments
   - End with forward-looking vision

3. **Authority Building**
   - Cite credible sources
   - Include data and statistics
   - Reference industry examples
   - Add expert quotes if available

4. **Tone Calibration**
   - Professional but accessible
   - Opinionated but balanced
   - Authoritative but humble

5. **Output**
   - Save to: `workspace/memory/night-shift/content/thought-leadership-[topic]-YYYY-MM-DD.md`

### Workflow 3: Industry Analysis Piece
1. **Data Gathering**
   - Collect metrics and trends
   - Research market developments
   - Monitor competitor activities
   - Gather expert opinions

2. **Analysis Framework**
   - Define scope and parameters
   - Establish analytical approach
   - Plan comparison points
   - Note limitations

3. **Writing**
   - Present data clearly
   - Interpret significance
   - Draw conclusions
   - Make predictions

4. **Visual Planning**
   - Note charts/graphs needed
   - Suggest data visualization
   - Plan image placement

5. **Output**
   - Save to: `workspace/memory/night-shift/content/analysis-[topic]-YYYY-MM-DD.md`

### Workflow 4: Editorial Newsletter Content
1. **Theme Selection**
   - Align with editorial calendar
   - Consider current events relevance
   - Review reader interests

2. **Content Curation**
   - Gather article summaries
   - Write original commentary
   - Include industry news
   - Add helpful resources

3. **Voice Development**
   - Conversational and engaging
   - Expert but approachable
   - Consistent personality

4. **Section Development**
   - Opening essay/letter
   - Curated links with commentary
   - Original insights
   - Closing/next week preview

5. **Output**
   - Save to: `workspace/memory/night-shift/content/newsletter-content-YYYY-MM-DD.md`

- Insufficient research data available
- Conflicting information requiring clarification
