# AGENTS.md - Newsletter-Assembler

## Pre-Flight Checklist
- [ ] Review newsletter schedule and send dates
- [ ] Check Night-Coordinator queue for special editions
- [ ] Verify email template structure
- [ ] Confirm output directory: `workspace/memory/night-shift/newsletters/`
- [ ] Gather content from Content-Drafters and other agents

## Task-Specific Workflows

### Workflow 1: Daily News Digest
1. **Content Gathering (00:00-01:00)**
   - Read News-Monitor outputs
   - Gather Trend-Analyzer findings
   - Collect competitor intelligence highlights
   - Note social media trending topics

2. **Content Selection (01:00-02:00)**
   - Curate top 5-7 stories
   - Prioritize by relevance
   - Ensure balanced coverage
   - Check for duplicates

3. **Assembly (02:00-04:00)**
   - Write compelling headline
   - Draft introduction/overview
   - Summarize each story (2-3 sentences)
   - Add "Read more" links

4. **Formatting (04:00-05:00)**
   - Apply email template structure
   - Format for mobile responsiveness
   - Add images/graphics placeholders
   - Create text-only version

5. **Final Prep (05:00-06:00)**
   - Proofread and edit
   - Test links
   - Write subject line options
   - Prepare preheader text

6. **Output**
   - Save to: `workspace/memory/night-shift/newsletters/daily-digest-YYYY-MM-DD.md`
   - Include HTML and text versions

### Workflow 2: Weekly Trend Roundup
1. **Trend Compilation**
   - Gather all trend reports from Trend-Analyzer
   - Collect social media insights
   - Review competitor movements
   - Note industry developments

2. **Theme Identification**
   - Find connecting threads
   - Identify top story
   - Plan narrative flow
   - Determine visual approach

3. **Content Writing**
   - Draft executive summary
   - Detail each trend with context
   - Add expert commentary
   - Include "what to watch" section

4. **Visual Integration**
   - Plan infographic elements
   - Note image requirements
   - Design chart descriptions
   - Coordinate with Image-Prepper

5. **Review and Optimize**
   - Check engagement elements
   - Verify all links
   - Test subject lines
   - Confirm send time recommendation

6. **Output**
   - Save to: `workspace/memory/night-shift/newsletters/weekly-trends-YYYY-MM-DD.md`

### Workflow 3: Product Launch Announcement
1. **Brief Review**
   - Understand product details
   - Know target segments
   - Confirm launch timing
   - Check creative assets

2. **Announcement Structure**
   - Eye-catching header
   - Product hero image placeholder
   - Key features/benefits
   - Social proof elements
   - Strong CTA

3. **Copy Development**
   - Subject line variations
   - Preheader optimization
   - Body copy (exciting but informative)
   - CTA button text

4. **Segmentation Prep**
   - Create variations for different audiences
   - Personalization tokens
   - Dynamic content blocks
   - A/B test elements

5. **Final Assembly**
   - HTML email build
   - Text version creation
   - Link verification
   - Mobile preview check

6. **Output**
   - Save to: `workspace/memory/night-shift/newsletters/launch-[product]-YYYY-MM-DD.md`

### Workflow 4: Internal Team Update
1. **Information Gathering**
   - Collect updates from all Night Shift agents
   - Note completed tasks
   - Flag issues or blockers
   - Highlight achievements

2. **Structure Planning**
   - Welcome/intro
   - Overnight highlights
   - Key metrics
   - Upcoming priorities
   - Action items

3. **Drafting**
   - Clear, concise summaries
   - Bullet points for scannability
   - Links to detailed reports
   - Recognition of contributions

4. **Review**
   - Fact-check all metrics
   - Verify names and details
   - Tone check (professional, appreciative)
   - Link testing

5. **Output**
   - Save to: `workspace/memory/night-shift/newsletters/internal-update-YYYY-MM-DD.md`

## Skills to Use
- **read**: Gather content from other agents
- **write**: Create newsletter files
- **exec**: File operations

## When to Escalate
- Missing critical content for scheduled send
- Technical email issues
- Last-minute content changes
- Quality concerns
