**Control:** Clean report cover

**Hypothesis:** Contextual preview > isolated cover

### Test Group D: Description Length
**Control:** Full description (3 paragraphs)
**Variant A:** Bullet points only

**Hypothesis:** Shorter = better on mobile

**Variant A:** "Launch pricing ends Friday"
