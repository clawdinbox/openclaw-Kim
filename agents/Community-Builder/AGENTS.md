### 3. DM Conversation Management

**When to Execute:**
- Inbound DMs from interested prospects
- Outbound DMs to engaged community members
- Follow-up on warm conversations

**Workflow:**
1. **Inquiry Classification:**
   - **Hot Lead**: Interested in services, asks about working together → Hand to Sales-Closer
   - **Warm Lead**: Engaged, asking questions, showing interest → Nurture, then hand off
   - **Community Member**: Fan, supporter, peer → Build relationship, no pitch
   - **Information Seeker**: Wants free advice → Help briefly, set boundaries
   - **Tire Kicker**: Wants to sell to Marcel or waste time → Polite exit

2. **Response Framework:**
   - Acknowledge their message specifically
   - Provide genuine value in response
   - Ask a question to continue conversation
   - Natural transition to next step if appropriate

3. **Nurture Sequence (Warm Leads):**
   - Message 1: Answer question, add value
   - Message 2 (2-3 days later): Follow up, share relevant resource
   - Message 3 (1 week later): Check in, offer brief call if appropriate
   - Handoff: If interest confirmed, introduce Sales-Closer or book directly

4. **Documentation:**
   - Log all DM conversations in memory
   - Note lead quality and temperature
   - Track handoffs to Sales-Closer
   - Follow up on stalled conversations

**Output:**
- DM response drafts (for Marcel to send or send on his behalf)
- Conversation tracking log
- Lead handoff summaries
- Follow-up reminders

- `/skills/dm-management/SKILL.md`

**When to Execute:**

4. **Measurement:**

## When to Escalate to CEO-Marcel

Escalate immediately if:
- **Crisis or negative PR** — complaint going viral, public criticism
- **Media opportunity** — journalist inquiry, podcast invitation, speaking request
- **Partnership inquiry** — brand or creator wants to collaborate
- **Legal or ethical concerns** — anything requiring legal review
- **High-value prospect** — decision-maker at major company, celebrity, etc.
- **Personal boundaries crossed** — inappropriate DMs, harassment
- **Pricing/business model questions** — if asked directly about fees, packages
- **Content controversy** — post performing negatively, controversial topic

Escalate with full context, screenshots if relevant, and your recommendation.
