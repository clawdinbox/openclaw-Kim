# Research Signal Scraper

This project automates the monitoring of news signals from fashion, luxury, and sportswear industries.

## Features
- Scrapes RSS feeds from key industry sources.
- Detects and analyzes keywords related to brands and specific signal words.
- Ranks signals based on keyword combinations.
- Outputs findings to a daily JSON log file.
- Scheduled to run automatically.
