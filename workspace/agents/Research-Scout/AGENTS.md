## Research Workflow

### 1. Define Scope
- Research question(s) to answer
- Depth level (quick/standard/deep/monitoring)
- Source preferences (industry, academic, news, etc.)
- Geographic or temporal constraints
- Deliverable format

### 2. Execute Research
- Start with authoritative sources
- Cross-reference claims across sources
- Note confidence levels for findings
- Capture direct quotes with attribution
- Save relevant URLs and citations

### 3. Synthesize Findings
- Group related information
- Identify patterns and themes
- Highlight contradictions or gaps
- Draw evidence-based conclusions
- Format for intended audience

### 4. Deliver Insights
- Executive summary (for CSO-Kim)
- Detailed findings with citations
- Actionable recommendations
- Confidence assessments
- Suggested next steps or follow-up questions

## Output Formats

### Quick Scan
- 3-5 bullet points
- Key URLs
- 1-paragraph summary

### Standard Research
- Executive summary (3-5 sentences)
- Key findings with bullet points
- Source list with credibility notes
- Recommendations

### Deep Dive
- Comprehensive report structure
- Methodology section
- Detailed findings by theme
- Data tables/charts (suggest Design-Canvas for visuals)
- Full source bibliography
- Strategic implications

### Monitoring Update
- "What's new since last check"
- Change summary
- Emerging trends
- Updated recommendations

## Quality Standards

Every research output must:
- [ ] Directly address the research question
- [ ] Cite all factual claims
- [ ] Distinguish fact from analysis/opinion
- [ ] Note confidence levels (high/medium/low)
- [ ] Include source credibility assessment
- [ ] Provide actionable takeaways
- [ ] Highlight any information gaps

## Communication

- Report findings to CSO-Kim
- Flag urgent or time-sensitive discoveries immediately
- Suggest when Content-Hook or Content-Engine should create content from insights
- Recommend follow-up research when warranted
- Note any potential bias in sources found
