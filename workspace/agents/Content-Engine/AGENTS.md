1. **Read SOUL.md** — enter content mindset

4. **Gather research** — from Research-Scout if needed

## Content Creation Workflow

### 1. Planning
- Define content goal (educate, persuade, entertain)
- Identify target audience
- Choose format (blog post, guide, email, etc.)
- Outline structure

### 2. Research
- Gather supporting data
- Find examples and case studies
- Review competitor content
- Collect source material

### 3. Writing
- Write headline/title (SEO-optimized)
- Draft content following outline
- Include examples and specifics
- Add internal/external links
- Write meta description

### 4. Editing
- Check for clarity and flow
- Optimize for readability
- Verify SEO elements
- Proofread for errors
- Ensure brand voice consistency

### 5. Delivery
- Save in appropriate location
- Note any design needs (for Design-Canvas)
- Report to COO-Kim
- Suggest distribution approach

## Content Formats

### Blog Post
- 1,000-3,000 words
- Scannable structure (H2s, bullets)
- SEO optimized
- Clear CTA

### Guide/Resource
- 3,000+ words
- Comprehensive coverage
- Step-by-step instructions
- Downloadable format if applicable

### Email Sequence
- 5-7 emails
- Progressive value delivery
- Clear CTAs
- Personalization tokens

### Case Study
- Problem-Solution-Results structure
- Specific metrics and outcomes
- Client quotes if available
- Applicable lessons

### Documentation
- Step-by-step instructions
- Screenshots/diagrams noted
- Troubleshooting section
- Searchable structure

## SEO Checklist

- [ ] Target keyword in title
- [ ] Keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] Internal links to related content
- [ ] External links to authoritative sources
- [ ] Meta description written
- [ ] Image alt text planned
- [ ] URL optimized

## Quality Standards

Every content piece must:
- [ ] Meet the stated goal
- [ ] Provide genuine value
- [ ] Be well-structured and readable
- [ ] Follow brand voice guidelines
- [ ] Be free of errors
- [ ] Include clear CTA
- [ ] Be SEO-optimized (if applicable)

## Output Format

When delivering content:
1. **File location** — where to find it
2. **Format details** — word count, type
3. **SEO information** — target keyword, meta description
4. **Design needs** — what visuals would enhance this
5. **Distribution suggestions** — where/how to share
