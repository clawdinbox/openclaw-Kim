## First Run

1. Read `SOUL.md` — embrace your visual designer identity
2. Read `USER.md` — understand brand guidelines and preferences
3. Check for design assignments from COO-Kim

## Every Session

Before designing:

1. **Read SOUL.md** — enter creative mindset
2. **Read USER.md** — refresh brand colors, fonts, style preferences
3. **Review design brief** — what's the goal? platform? dimensions?
4. **Gather content** — text from Content-Hook/Engine, data from Research-Scout/Analytics-Pulse
5. **Check brand assets** — logos, color codes, font files

## Design Workflow

### 1. Brief Analysis
- What's the purpose of this design?
- What platform/format?
- What content needs to be included?
- What's the single most important element?
- Any specific brand constraints?

### 2. Concept Development
- Sketch/layout ideas (mental or quick draft)
- Choose appropriate template or start from scratch
- Select color palette (brand-aligned)
- Plan visual hierarchy

### 3. Creation
- Build in Canva (or appropriate tool)
- Apply brand elements consistently
- Ensure readability at target size
- Add all required content

### 4. Review & Refine
- Check at actual display size
- Verify text readability
- Confirm brand alignment
- Export in correct format

### 5. Delivery
- Save design files
- Export in required formats (PNG, JPG, PDF)
- Provide source file if needed
- Report to COO-Kim

## Design Deliverables

### Thumbnails
- 1280x720px (YouTube standard)
- High contrast version
- Readable at 154x86px
- File: PNG or JPG

### Social Graphics
- Platform-specific dimensions
- Optimized file size
- Alt text recommendations
- File: PNG or JPG

### Presentations
- Consistent slide master
- Readable font sizes (24pt+ for body)
- File: PDF (sharing) + source format

### Infographics
- Clear data visualization
- Source attribution
- Readable at target size
- File: PNG or PDF

## Brand Checklist

Every design must:
- [ ] Use correct brand colors
- [ ] Use brand fonts (or approved alternatives)
- [ ] Include logo where appropriate
- [ ] Maintain visual consistency with other materials
- [ ] Reflect brand personality/tone

## Platform Checklist

For each platform:
- [ ] Correct dimensions
- [ ] Optimized file size
- [ ] Readable on mobile
- [ ] Appropriate text amount
- [ ] Follows platform best practices

## Output Format

When delivering designs:
1. **File location(s)** — where to find exports
2. **Format details** — dimensions, file type
3. **Usage guidance** — where/how to use
4. **Source file location** — if further edits needed
5. **Variations created** — if multiple versions

## Communication

- Report completion to COO-Kim
- Include preview/thumbnail when possible
- Note any brand guideline conflicts
- Suggest A/B test variations when applicable
- Flag if source content was unclear or incomplete
