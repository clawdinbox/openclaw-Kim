# X/Twitter Scraping Alternativen zu Apify

## Option 1: Nitter (Kostenlos)
**Pros:**
- Kostenlos, Open Source
- Keine API-Keys
- Einfaches HTML Scraping

**Cons:**
- Instabil (wird oft blockiert)
- Keine Garantie auf Verfügbarkeit
- Rate limiting unvorhersehbar

## Option 2: X API Free Tier
**Pros:**
- Offizielle API
- Zuverlässig

**Cons:**
- Nur 500 Posts/Monat (zu wenig für uns)
- Höheres Tier: $100+/Monat

## Option 3: RapidAPI Twitter-Endpunkte
**Pros:**
- Günstiger als Apify (~$10-20/Monat)
- Viele verschiedene Provider
- Gute Rate Limits

**Cons:**
- Qualität variiert
- Manche sind instabil

## Option 4: Puppeteer/Playwright (Self-Built)
**Pros:**
- Komplette Kontrolle
- Kostenlos (außer Server-Kosten)

**Cons:**
- Hoher Maintenance-Aufwand
- X blockiert automatisierte Browser
- Zeitintensiv

## Option 5: Manuelle Kuratierung (Empfohlen für Start)
**Pros:**
- Höchste Qualität
- Direkte Relevanz-Filterung
- Keine technischen Probleme

**Cons:**
- Zeitaufwändig
- Nicht echtzeit
